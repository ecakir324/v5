# Privora v5

GitHub Actions ile Android Studio kurmadan debug APK üretmek için sadeleştirilmiş sürüm.

Özellikler:
- PIN ekranı
- Foto/video import
- Uygulama özel klasörüne hızlı gizleme
- Grid galeri
- Video izleme
- Video kapak saniyesi seçme
- Silme
- Opsiyonel AES-GCM şifreleme

Repo kökünde görünmesi gerekenler:
- app/
- .github/
- build.gradle
- settings.gradle
